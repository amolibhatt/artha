# Artha AI: Personal CFO & Goal-Driven Wealth Engine

**Artha AI** is a McKinsey-level financial strategist designed to transform raw transaction data into a high-octane capital allocation system. It moves beyond simple "budgeting" into the realm of dynamic cash flow optimization and capital protection.

## 🏛 The Global Framework

Artha operates on five core operating principles:
1. **Dynamic over static**: Adapts instantly to new income and expense vectors.
2. **Sequence over parallel**: Prioritizes debt liquidation and capital reserves intelligently.
3. **Cash flow realism**: Focuses on actual liquidity rather than theoretical net worth.
4. **Capital protection**: Ensures short-term goals are never compromised by market volatility.
5. **Clear trade-offs**: Explicitly visualizes the cost of discretionary spending.

## 🚀 Key Features

### 1. Capital Allocation Dashboard
A high-level audit of your capital deployment across three primary vectors:
*   **Operating Burn**: Real-time monitoring of lifestyle maintenance costs.
*   **Capital Reserve**: Tracking the surplus required for long-term stability.
*   **Goal Momentum**: Visualizing the velocity of your wealth-building targets.

### 2. Survival Metrics
*   **Financial Runway**: A "days-to-zero" survival metric if income stops today.
*   **Daily Discretionary Allowance**: A safe daily spending limit calculated *after* accounting for all goal contributions.

### 3. Capital Leakage Audit
A sophisticated breakdown of expenses categorized by strategic impact, allowing you to identify and plug efficiency risks immediately.

### 4. Goal-Oriented Wealth
Dynamic tracking for Savings, Investments, and Debt Liquidation with automated progress forecasting.

## 🛠 Tech Stack

*   **Frontend**: React 18, TypeScript, Tailwind CSS
*   **Animations**: Framer Motion
*   **Charts**: Recharts (D3-based)
*   **Backend**: Node.js / Express (SPA optimized)
*   **Database/Auth**: Firebase Firestore & Google Auth
*   **AI Engine**: Google Gemini API (Financial Analysis)

## 📦 Setup & Installation

### Prerequisites
*   Node.js (v18+)
*   Firebase Project

### Installation
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/artha-ai.git
    cd artha-ai
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Environment Variables**:
    Create a `.env` file in the root directory and add your credentials:
    ```env
    GEMINI_API_KEY=your_gemini_api_key
    VITE_FIREBASE_API_KEY=your_key
    VITE_FIREBASE_AUTH_DOMAIN=your_domain
    VITE_FIREBASE_PROJECT_ID=your_id
    VITE_FIREBASE_STORAGE_BUCKET=your_bucket
    VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
    VITE_FIREBASE_APP_ID=your_app_id
    ```

4.  **Run Development Server**:
    ```bash
    npm run dev
    ```

5.  **Build for Production**:
    ```bash
    npm run build
    npm start
    ```
